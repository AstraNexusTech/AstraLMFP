## ⚡ 简介 (Introduction)

**Astra LMFP** 是一款专为 Minecraft 打造的自动化极速联机客户端。它打破了传统联机的繁琐步骤，将核心流程极致压缩：
**自动获取本地端口 ➔ 自动完成内网穿透 ➔ 自动发布至联机大厅**，所有操作一气呵成。

本分支定位于**客户端 UI 深度进化与跨平台重构**：
* 🎨 **UI 彻底重构**：基于 Google 官方 **Material Design 3 (M3)** 规范，打造通透、优雅、具备呼吸感的高级交互界面。
* 🖥️ **全平台适配 & 兼容**：得益于 **Flet (Flutter Core)** 引擎，不仅支持 Windows 10/11，还提供了更广泛的平台适配可能，并针对老旧系统做出了优化。
* ⚡ **异步性能调优**：对原版后端的阻塞逻辑进行了多线程与异步解耦，拒绝界面卡死，运行更加丝滑。

## 🛠️ 客户端配置 (Client Configuration)

Astra 分支仅提供客户端外壳与核心对接逻辑，定位于 `LMFP` 目录，配置十分简单：

1. 打开核心代码文件 `main.py`。
2. 找到 `apis` 变量。
3. 修改为你自己部署或获取到的通用服务端 API 地址即可。

> 🔒 **关于服务端 API 的说明**
> 本分支**不提供任何服务端的 API 部署文档与说明**。本分支的 API 与上游源代码 API 保持通用，如需自行架设 API 层，请移步参考原作者源代码进行逆向或分析。
